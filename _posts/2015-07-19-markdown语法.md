- ###标题
Markdown 支持两种标题的语法，Setext 和 atx 形式。Setext 形式是用底线的形式，利用 = （最高阶标题）和 - （第二阶标题），Atx 形式在行首插入 1 到 6 个 # ，对应到标题 1 到 6 阶。

一阶标题
===
二阶标题
---
#一阶标题
##二阶标题

- ###代码

行的开头空4个空格，表示程序代码，例如：  
C#:

    public class Blog
    {
         public int Id { get; set; }

Javascript:

    function fib(n) {
        var a = 1, b = 1;
        var tmp;
        while (--n >= 0) {
            tmp = a;
            a += b;
           b = tmp;
        }
        return a;
    }

    document.write(fib(10));
